# 🌐 My Personal Website

This is my personal developer website built with plain HTML, CSS, and a bit of JavaScript.

It serves as a portfolio to showcase my projects, skills, and background as a software developer.

---

## 🚀 Live Website

👉 https://aschmidtphil.github.io/my-webseite  

---

## 🧱 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages (hosting)

No frameworks or build tools — intentionally kept simple to focus on performance, clarity, and fundamentals.

---

## 🎯 Purpose of this Project

This website was built to:

- Practice and strengthen core web development fundamentals
- Create a clean and minimal developer portfolio
- Learn how static websites are deployed using GitHub Pages
- Improve structure, design, and UX without frameworks
